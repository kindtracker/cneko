# Cneko

## About
Cneko is a statically-typed systems language inspired by C3 and Lua. It is a JIT programming language and focuses on simplicity.

> [!WARNING]
> Cneko is in early development. The language is subject to significant changes. Breaking changes expected.

## Quick Start
```c
import std:io;

fn main() {
  io:printfn("Hello, Cneko!");
}
```
